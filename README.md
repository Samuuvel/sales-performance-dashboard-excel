# Sales Performance Excel Dashboard

## Project Overview

This project analyzes sales performance data using Microsoft Excel to generate business insights related to revenue trends, profit margins, regional performance, and product analysis.

---

## Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Conditional Formatting
* VLOOKUP
* IF Functions
* SUMIFS
* Dashboard Design

---

## Dashboard Features

* KPI Cards
* Revenue Tracking
* Profit Margin Analysis
* Region-wise Sales Performance
* Product Category Analysis
* Monthly Sales Trends
* Interactive Filters & Slicers

---

## Analysis Performed

* Sales trend analysis
* Product performance analysis
* Regional comparison
* Profitability analysis
* Revenue growth tracking

---

## Key Business Insights

* Identified top-performing products
* Detected underperforming regions
* Analyzed monthly revenue fluctuations
* Improved visibility into business KPIs
